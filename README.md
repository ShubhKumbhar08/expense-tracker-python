# Expense Tracker (Python)

This is a simple command-line based Expense Tracker application built using Python.
It helps users record daily expenses, manage budgets, and analyze spending category-wise.

## Features
- Add new expenses
- View all recorded expenses
- Category-wise expense summary
- Set and check budget
- Search expenses by category, amount, or description

## How to Run
1. Make sure Python is installed on your system
2. Run the following command:

```bash
python expense_tracker.py
```

## Technologies used
- Python
- Lists
- Dictionaries
- Loops and conditional Statements
